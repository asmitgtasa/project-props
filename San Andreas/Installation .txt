Project Props 
-----------------------------

Requirements
---------------------------
1.Limit Adjusters.I suggest Fastman92 Limit Adjuster.

For Downloads Goto http://fastman92.com/

2.Garage Extended

Our mod added some extra garages in the city of  San Andreas.So it is necessary to hack the garage limit of SA.
Note:Only needed if you wanna use the garages.

For Downloads FLA Comportable build got to.
https://github.com/goodidea82/GarageExtender



Installation:

Normal Installation:

1.Paste the data Folder in your game  directory.
2.Replace the gta.dat included in our files.


Note:Not for any mod projects which has gta.dat modified.For example SAxVCxLC.
_______________________

Alternative Installation:

If you got any previous map mods installed try this one.

1.Paste the data Folder in your game directory.
2.Open gta.dat with notepad or any text editor.
3.Go to the Ipl loading section.
4.Search For this line
     IPL DATA\MAPS\audiozon.ipl
5.Then paste  This Line After It
      #Project Props
IPL DATA\MAPS\Project Props\SApropsAnglePine.ipl
IPL DATA\MAPS\Project Props\SApropsLV.ipl
IPL DATA\MAPS\Project Props\SapropsSF.IPL
IPL DATA\MAPS\Project Props\SApropsLS.ipl
6.Then Save.
7.Finished.

_______________________
Mod Loader

1.Paste the Data Folder in Mod Loader directory.
2.Finished
